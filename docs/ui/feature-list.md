# 機能一覧

本ドキュメントはリクエコの機能をユーザータイプ別にまとめたものです。

*最終更新: 2026-02-12*

---

## ユーザータイプ

| タイプ | 説明 | 認証方法 |
|--------|------|----------|
| 求職者 | 求人を探し応募する一般ユーザー | 認証不要 |
| Admin | システム全体を管理する管理者 | Google OAuth |
| 会社ユーザー | 自社の求人・採用を管理する企業担当者 | メール/パスワード |

---

## 1. 求職者向け機能

### 求人検索・閲覧

| 機能 | 説明 | 画面 |
|------|------|------|
| 求人一覧 | 全求人の検索・フィルター・一覧表示 | jobs.html |
| 求人詳細 | 個別求人の詳細情報閲覧 | job-detail.html |
| 勤務地検索 | 都道府県別の求人検索 | location.html |
| 会社一覧 | 掲載企業の一覧表示 | company.html |

### 会社・求人ページ

| 機能 | 説明 | 画面 |
|------|------|------|
| 会社採用ページ | 企業ごとのカスタム採用ページ閲覧 | company-recruit.html |
| 求人LP | 求人ごとのランディングページ閲覧 | lp.html |
| デザインパターン | 5種類のLPデザイン（standard/modern/classic/minimal/colorful） | lp.html |

### 応募・コンタクト

| 機能 | 説明 | 画面 |
|------|------|------|
| Web応募 | 応募フォームからの応募 | lp.html (モーダル) |
| 電話応募 | ワンクリック電話発信（モバイル） | lp.html |
| LINE応募 | LINEでの問い合わせ | lp.html |
| 応募完了通知 | 応募後の確認画面表示 | lp.html |

### マイページ

| 機能 | 説明 | 画面 |
|------|------|------|
| お気に入り | 気になる求人の保存（localStorage） | mypage.html |
| 応募履歴 | 過去の応募履歴確認 | mypage.html |
| 閲覧履歴 | 閲覧した求人の履歴 | mypage.html |

### その他

| 機能 | 説明 | 画面 |
|------|------|------|
| ガイドページ | 仕事探し・面接・働き方ガイド | guide-*.html |
| 会社概要 | サービス運営会社情報 | about.html |
| 利用規約 | サービス利用規約 | terms.html |
| プライバシーポリシー | 個人情報取り扱い | privacy.html |

### 会社管理

### 求人管理

## 3. 会社ユーザー向け機能

### 閲覧可能

| 機能 | Admin機能との違い |
|------|------------------|
| 概要ダッシュボード | 自社データのみ表示 |
| 求人管理 | 自社求人のみ |
| LP設定 | 自社LPのみ |
| 採用ページ設定 | 自社のみ |
| 応募者管理 | 自社応募者のみ |
| 会社ユーザー管理 | 自社ユーザーのみ |

### アナリティクス（制限あり）

| タブ | アクセス |
|------|---------|
| 採用管理 | × 非表示 |
| 企業別 | × 非表示 |
| LP/採用ページ | ○ 自社データのみ |
| コンバージョン | ○ 自社データのみ |
| トラフィック | ○ 自社データのみ |
| トレンド | ○ 自社データのみ |

### 利用不可

| 機能 | 理由 |
|------|------|
| 会社管理 | Admin専用 |
| 全社求人一覧 | Admin専用 |
| 設定 | Admin専用 |
| 企業別分析 | Admin専用 |
| 採用管理分析 | Admin専用 |

### 追加機能

| 機能 | 説明 |
|------|------|
| パスワード変更 | 自身のパスワード変更 |
| プロフィール編集 | 表示名等の変更 |

## 外部連携機能

### 広告プラットフォーム連携

| プラットフォーム | 機能 | 設定場所 |
|-----------------|------|----------|
| TikTok Pixel | ViewContent/コンバージョン追跡 | LP設定 |
| Google Ads | コンバージョンタグ | LP設定 |
| Meta Pixel | PageView/ViewContent/Lead/Contact | LP設定 |
| LINE Tag | ページビュー/コンバージョン | LP設定 |
| Microsoft Clarity | セッション録画/ヒートマップ | LP設定 |

### API連携

| サービス | 用途 |
|---------|------|
| Google Analytics 4 | アクセス解析データ取得 |
| Firebase Authentication | ユーザー認証 |
| Firestore | データストレージ |
| Cloud Functions | バックエンドAPI |
| Google Calendar API | 面談スケジュール連携 |
| SendGrid | メール送信 |
